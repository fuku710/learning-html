# HTML タグあれこれ

HTML タグではたくさんのタグがあります。
適切なタグを使いましょう！

## h タグ

見出しに使うタグ、h1 から h5 まであるのでうまく使い分けよう。

```html
<body>
  <h1>見出し1</h1>
  <h2>見出し2</h2>
  <h3>見出し3</h3>
  <h4>見出し4</h4>
  <h5>見出し5</h5>
</body>
```

## p タグ

paragrah(=段落)を意味するタグ。ある程度の量の文章を囲んだりする。

```html
<body>
  <p>吾輩は猫である。</p>
</body>
```

## div タグ

div タグ自体は特に意味はない。大体 CSS を使う際にグループとして囲うために使う。

## span タグ

div タグ同様に特に意味はない。span タグの場合は文章中の単語のみというふうに一部分だけ囲うときに使う。
