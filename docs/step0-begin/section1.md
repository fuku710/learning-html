# HTML の基本

皆さんが PC やスマートフォンなどの WEB ブラウザから見るページは HTML(HyperTextMarkupLanguage)という言語で記述されています。  
任意のディレクトリで以下のような index.html を作成して自分の好きなブラウザで開いてみましょう。

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Webページのタイトル</title>
    <meta charset="utf-8" />
  </head>
  <body>
    <h1>見出し</h1>
    <p>こんにちは</p>
  </body>
</html>
```
